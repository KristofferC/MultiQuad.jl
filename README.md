# MultiQuad.jl
MultiQuad.jl is a convenient interface to perform 1d, 2d and 3d numerical integrals.
It uses [QuadGK](https://github.com/JuliaMath/QuadGK.jl) and [Cuba](https://github.com/giordano/Cuba.jl) as back-ends.
